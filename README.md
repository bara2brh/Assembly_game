# Assembly: Endgame 🎮

A fun word-guessing game built with React where players must guess the hidden word before all programming languages are eliminated and Assembly takes over.

## Features

* Random word generation
* Interactive on-screen keyboard
* Win and lose game states
* Confetti celebration on victory
* Dynamic farewell messages for eliminated languages
* Accessible UI with ARIA support
* Responsive design

## How to Play

1. Guess letters using the on-screen keyboard.
2. Correct letters are revealed in the hidden word.
3. Wrong guesses eliminate programming languages.
4. Guess the entire word before all languages are lost.
5. Start a new game and try again when the game ends.
